Live Setup

直播预设

进入直播前先补一下这次的基础信息。这里只记录本次直播的主题和挂载来源，后续进入直播会直接沿用这里的预设。

开启后，会记录本次直播需要带入 Bubble 的基础设定。

可选勾选本次直播要带入的世界书；不勾选具体条目时，也会保留“已开启世界书”的状态。

Signing Setup

签售预设

进入签售前先补一下这次的基础信息。签售会按单粉丝轮转进行，每位粉丝都会单独计时。

可选勾选本次签售要带入的世界书；不勾选具体条目时，也会保留“已开启世界书”的状态。

Privacy Cover

白名单

Confirmed

全部白名单预览

所有已确认进入白名单的词条都会同步到云端白名单表；支持直接改名、删除，改动会自动同步。

Exclude

扫描排除词

加到这里的词不会再被自动隐私覆盖；支持直接改名、删除，改动会自动生效。

Recent Hits

最近实际屏蔽词

这里展示最近请求里真实命中的屏蔽词；支持直接改名后加入白名单或加入扫描排除。

扫描只会把新词放到待确认区；已在白名单里的词会自动跳过。每条候选都可以直接加入或删除。

Manual Add

新增白名单

手动新增现在按结构化白名单录入。请先确认词条类型并填写分组；实际占位符由系统自动复用或生成。
